# bitbox-birdy

A clone of flappy bird game for bitbox console http://bitboxconsole.blogspot.com.ar/ .

Pixel art graphics obtained from https://opengameart.org of the user "LD" or "Lucid Design" https://opengameart.org/users/ld .
